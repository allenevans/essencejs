registerByOptions

Config object passed for registering files from the file system.
Type:
  • Object
Properties:
Name Type Description
namespace string namespace to register with.
cwd string The current working directory in which to search. Defaults to process.cwd().
root string The place where patterns starting with / will be mounted onto. Defaults to path.resolve(options.cwd, "/")
nosort boolean Don't sort the results.
sync boolean Perform a synchronous glob search.
nobrace boolean Do not expand {a,b} and {1..3} brace sets.
noglobstar boolean Do not match ** against multiple filenames. (Ie, treat it as a normal * instead.)
noext boolean Do not match +(a|b) "extglob" patterns.
nodir boolean Do not match directories, only files.
